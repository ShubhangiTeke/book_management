# 📚 Book Management System (Java)

##  About the Project

This is a console-based Book Management System developed in Java as part of my Object-Oriented Programming (OOP) learning and college assignment.

The project allows users to manage book records through a menu-driven interface while practicing core Java concepts.

---

##  Features

- 📖 Add a new book
- 🔍 Search a book by Book ID
- 📅 Search books by Publication Year
- 📋 Display all books

---

##  Technologies Used

- Java
- Object-Oriented Programming (OOP)

---

##  Concepts Practiced

- Classes and Objects
- Constructors
- Encapsulation
- Arrays
- Getters and Setters
- Menu-Driven Programming

---

##  Project Structure

```
Program.java
│
├── Date Class
├── Book Class
└── Main Program
```

---

##  How to Run

1. Clone this repository.
2. Open the project in Eclipse, IntelliJ IDEA, or any Java IDE.
3. Run the `Program.java` file.
4. Follow the menu displayed in the console.
